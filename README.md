# ✈️ AirSentiment – Airline Tweet Sentiment Analysis

AirSentiment is a Machine Learning based web application that predicts the sentiment of airline-related tweets as **Positive**, **Neutral**, or **Negative** using Natural Language Processing (NLP) techniques.

The project is built using:
- Python
- Streamlit
- Scikit-learn
- NLTK
- TF-IDF Vectorization
- Multinomial Naive Bayes

---

# 🚀 Features

- Predict sentiment of custom airline tweets
- Detect Positive, Neutral, and Negative sentiments
- Interactive Streamlit web interface
- Real-time prediction
- Sentiment distribution visualization
- Word cloud generation for each sentiment category
- NLP preprocessing using NLTK
- TF-IDF vectorization for feature extraction

---

# 🧠 Machine Learning Workflow

1. Dataset Loading
2. Text Preprocessing
3. Tokenization & Lemmatization
4. Stopword Removal
5. TF-IDF Vectorization
6. Model Training using Multinomial Naive Bayes
7. Sentiment Prediction
8. Visualization & Analysis
